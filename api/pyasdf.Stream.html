Stream¶

class pyasdf.Stream(shape, dtype, strides=None)[source]¶

Bases: pyasdf.tags.core.ndarray.NDArrayType

Used to put a streamed array into the tree.

Examples

Save a double-precision array with 1024 columns, one row at a time:

>>> from pyasdf import AsdfFile, Stream
>>> import numpy as np
>>> ff = AsdfFile()
>>> ff.tree['streamed'] = Stream([1024], np.float64)
>>> with ff.write_to('test.asdf') as ff:
...     for i in range(200):
...         ff.write_to_stream(np.array([i] * 1024, np.float64).tostring())
